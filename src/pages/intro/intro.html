O que é Ionic?


O Ionic Framework é um SDK de código aberto que permite aos desenvolvedores criar aplicativos móveis de alta qualidade com tecnologias da web como HTML, CSS e JavaScript.

O que é Ionic Cloud?


O Ionic Cloud é uma plataforma em nuvem para gerenciar e dimensionar aplicativos ônicos com serviços integrados, como notificações push, compilações nativas, autenticação de usuário e atualização ao vivo.
